== Description ==

This plugin is usable for RestroPress user who want to show their items in catalog mode.
We have added two types of view "List and Grid" which you can aplly to use as per your required UI.
Also we have options to view price for the items or not as per your requirement.

Apart of these current features, please suggest us more required feature so that we can imporove this product to be more user friendly.


== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "RestroPress - Catalog Mode".
3. Click install.
4. Click activate.
5. Once the plugin is installed then you can see RestroPress on the left navigation bar of WordPress Dashboard.
